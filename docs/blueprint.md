# **App Name**: Christmas Institute

## Core Features:

- Header Navigation: A persistent header with links to 'Home', 'Rules', and 'Register' sections of the site.
- Hero Section: A visually engaging section with Christmas-themed imagery and a concise overview of the institute.
- Rules Section: Displays the Christmas Institute registration rules, using clear and concise language.
- Registration Section: Form to register at the Christmas institute, taking first name, last name, and email.

## Style Guidelines:

- Primary color: Blue evoking a warm and inviting feeling of tradition.
- Background color: Off-white (#FAF9F6) providing a soft, clean backdrop that complements the primary color.
- Accent color: Cobalt blue to add contrast and holiday cheer, complementing the burgundy hue.
- Headline font: 'Playfair', a modern serif with an elegant, fashionable, high-end feel; Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality
- Use festive, hand-drawn style icons related to Christmas themes like ornaments, snowflakes, and presents.
- Use a clear and intuitive layout that guides users to important information.
- Subtle animations for button hovers and transitions.
